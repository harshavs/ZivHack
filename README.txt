*****************************
*Instructions to deploy app *
*****************************

* Project ZivHack is devloped using MongoDB, MongoJS, Node.js, Node.js Express, Jade and Bootstrap.
* DB Schema is in the file db/schema.txt as a sample JSON object for a product.
* Data is in the file db/data.js in the form of mongo shell commands. Execute this to get the data ready.
* Run app.js using node shell.
* Point to localhost:3000 in your browser.
* Home page shows all the products. And the category links on the top show the respective category products.
* Product page and cart are not implemented.
